Como conectar no Power BI — sem instalar nada
1
Obter Dados → Web → Avançado
2
Cole a URL da API no campo de URL
3
Clique em "Adicionar Cabeçalho" duas vezes e use os campos acima — copie cada nome e valor separadamente. O campo Authorization já vem com Bearer incluído — é só copiar e colar
4
Clique OK → Power Query abre o JSON → clique "Para a Tabela" → expandir colunas → Carregar

💡 Como usar no Power BI / Excel

Conectar direto pelo Power Query

  1. Clique em "Copiar URL" no arquivo desejado
  2. No Power BI Desktop: Página Inicial → Obter Dados → Web
  3. Cole a URL e clique OK
  4. O Power Query abre o arquivo automaticamente — escolha CSV como delimitador se necessário
⚡ Exercício avançado — Combinar todos os meses

Append Queries no Power Query

Para combinar todos os arquivos de uma vez, importe cada mês individualmente e use Página Inicial → Acrescentar Consultas no Power BI. É um ótimo exercício de Append Queries — ou use o XML Consolidado para ter tudo em um único arquivo navegável por ano.
