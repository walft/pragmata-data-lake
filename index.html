💡 Como usar no Power BI

Conectar direto pelo Power Query

  1. Clique em "Copiar URL" no arquivo desejado
  2. No Power BI Desktop: Página Inicial → Obter Dados → Web
  3. Cole a URL e clique OK
  4. O Power Query abre o arquivo automaticamente — escolha CSV como delimitador se necessário
⚡ Exercício avançado — Combinar todos os meses

Append Queries no Power Query

Para combinar todos os arquivos de uma vez, importe cada mês individualmente e use Página Inicial → Acrescentar Consultas no Power BI. É um ótimo exercício de Append Queries — peça ao professor para demonstrar essa técnica.
